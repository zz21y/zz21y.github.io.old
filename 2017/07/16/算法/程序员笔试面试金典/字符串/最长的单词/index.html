最长的单词

发表于 2017-07-16 | 分类于 程序员面试金典 ， 字符串

题目描述
有一组单词，请编写一个程序，在数组中找出由数组中字符串组成的最长的串A，即A是由其它单词组成的(可重复)最长的单词。
给定一个string数组str，同时给定数组的大小n。请返回最长单词的长度，保证题意所述的最长单词存在。

测试样例：
["a","b","c","ab","bc","abc"],6
返回：3
# 算法 # 字符串
检查一个单词是否为其他字符串的子串
找出文中这两个单词的最短距离
